### 👨‍💻 Sobre mim

- 🌍 Atualmente, sou **Desenvolvedor de Aplicações Geoespaciais na Orbty**.
- 🛰️ Anteriormente, atuei no desenvolvimento da **Plataforma TerraBrasilis (INPE)**, lidando com grandes volumes de dados ambientais.
- 🎓 **Formação:** Mestre em Computação Aplicada pelo **INPE** e Bacharel em Física Computacional pela **UFF**.
- 🔍 **Foco técnico:** Processamento de imagens de satélite, extração e análise de dados espaciais, e desenvolvimento de plataformas web voltadas para geoinformação.
- 💡 **Curiosidade:** Grande parte do meu trabalho diário de engenharia e análise espacial ocorre em repositórios privados, projetos acadêmicos e corporativos.
